# VideoVisualEffects

> **VideoVisualEffects** — простой инструмент для применения визуальных эффектов к видеофайлам.  
> Написан на Python с графическим интерфейсом на Tkinter и обработкой кадров через OpenCV.

---

## Возможности

- **11 встроенных эффектов:**

  | Эффект | Описание |
  |--------|----------|
  | Пиксели | Разбивает кадр на крупные блоки — ретро пиксель-арт |
  | Волны | Искажает строки/столбцы синусоидой с анимацией |
  | Полосы | Тёмные горизонтальные или вертикальные полосы |
  | Глитч | Случайные сдвиги блоков и каналов — эффект сбоя |
  | Хроматическая аберрация | Разводит RGB-каналы — радужные края |
  | Строки (Scanlines) | Затемнение каждой N-й строки — эффект CRT |
  | Квантизация цвета | Уменьшает кол-во оттенков — постеризация |
  | VHS | Сочетание шума, хроматики, строк и мерцания |
  | RGB Сдвиг | Независимый сдвиг каналов R, G, B |
  | Зеркало | Симметричное отражение кадра |
  | Неон | Светящиеся контуры на тёмном фоне |

- **Настраиваемые параметры** каждого эффекта через слайдеры прямо в окне
- **12 языков интерфейса:** RU, EN, UK, DE, FR, ES, PL, PT, IT, ZH, JA, KO
- **Настройки вывода:** FPS, масштаб разрешения (25–200%), кодек (mp4v / avc1 / XVID / MJPG)
- **Кнопка «Стоп»** — прерывает обработку в любой момент
- Прогресс-бар, статистика кадров и оценка оставшегося времени

## Требования

```
Python 3.8+
opencv-python
numpy
```

Установка зависимостей:

```bash
pip install opencv-python numpy
```

---

## Использование

1. Нажмите **Обзор...** и выберите исходное видео
2. Укажите путь для сохранения результата
3. Выберите эффект из выпадающего списка
4. Настройте параметры слайдерами
5. Нажмите **Начать обработку**

> ⚠️ Аудиодорожка **не переносится** в выходной файл.  
> Для кодека `avc1` (H.264) требуется установленный **ffmpeg**.
